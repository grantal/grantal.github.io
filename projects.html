Projects

These are repos for larger projects that I have worked on.

  • ti4calc - Calculates the odds of winning a battle in the board game Twilight Imperium. You can see it in action at grantal.net/ti4calc.
  • Backpack Builder - React app that lets you choose camping gear and then totals their weight. Running at grantal.net/backpack-builder.
  • WebGL Sandpiles - My senior thesis. It is a simulation of the Abelian sandpile model using Javascript and WebGL. Running at grantal.net/WebGL-Sandpiles.
  • TigerCompiler - A Compiler for the tiger language written in c++.
  • paneity - A website for students to get help with their homework.
  • solitaire - Multiplayer solitaire server and client I made for a class in C.
  • kronos - A Flask website for scheduling the oral defenses of students' senior theses.
  • vrfy - Django website for autograding cs homework assignements. Currently live at cs.reed.edu
